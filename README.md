# 🚀 webcode-play

Welcome to **webcode-play**! This is the core, lightweight testing playground engine behind the WebCode Studio ecosystem. 

Designed and built entirely from a smartphone interface, this single-file HTML utility allows young developers and students to code, compile, and preview web scripts instantly—even with zero setup or slow internet networks.

---

## ✨ Features

* 📱 **Mobile-First Responsive Layout:** Full screen optimization tailored specifically for small touch displays.
* ⚡ **Zero Setup Framework:** Open the file, select your language context, and hit run. No local ports required.
* 📉 **Ultra-Lightweight Core:** Clean single-file codebase under 74 lines of pure logical configuration.
* 🎨 **Dark Teal System Theme:** Comfortable IDE color palettes with variable control elements (`:root`).
* 🔒 **Isolated Output Sandbox:** Instant rendering via clean `srcdoc` iframes without storing hidden server logs.

---

## 🛠️ How It Works (The Logic)

1. Open the tool interface.
2. Select your language context from the language selector block (**HTML / CSS / JS**).
3. Type or paste your localized web code structures inside the coding zone.
4. Hit **▶ Run Code** to toggle the screen state and view the interactive canvas output instantly!

---

## 📂 Source Code Layout

The engine is contained completely within a simple, optimized HTML core layout:
* **UI/UX Aesthetics:** Pure modern layout mapping utilizing dark context tokens.
* **Callback Triggers:** Active screen state handlers managed smoothly via JavaScript `classList` injections.
